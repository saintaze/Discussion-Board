# Discussion-Board
A discussion board app where a user can signup/signup and start making comments to the feed.

### Features
+ Sliding Animation
+ Beautiful clean UI
+ Full SignUp/SignIn Auth Flow
+ Read/Create Comments
+ Comments and Users collection
+ Comments belongs to a User

##### Built With
+ HTML5
+ CSS3
+ React
+ Meteor
+ Semantic UI
